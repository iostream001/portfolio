# Norman Lyle Pono Portfolio

This is the latest portfolio website for Norman Lyle Pono, rebuilt as a
Vercel-ready Next.js app.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
- npm

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Validation

```bash
npm run lint
npm run build
```

## Vercel Import

- Repository: `iostream001/portfolio.git`
- Framework preset: `Next.js`
- Root directory: repository root
- Build command: `npm run build`
- Output directory: Vercel default for Next.js
- Environment variables: none required for v1
